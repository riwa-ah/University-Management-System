## University Management System
A desktop-based University Management System built using Java and Object-Oriented Programming (OOP) principles.
The system provides an easy-to-use graphical user interface (GUI) for managing students, courses, and faculties, with structured data handling and table-based visualization.

## Features
Student Management
Course Management
Faculty Management
Graphical User Interface (GUI)
  Tabbed navigation (Students / Courses / Faculty)
  JTable-based layouts for professional data display
  Rounded buttons and enhanced spacing
  Consistent teal-white theme for a modern experience

## How to Run

### Requirements
- Java JDK 8+
- IDE (IntelliJ, Eclipse, or VS Code) or Terminal

### Run Using IDE
1. Open the project in your IDE
2. Open Driver.java
3. Run the main method

### Run Using Command Line
cd src
javac *.java
java Driver
